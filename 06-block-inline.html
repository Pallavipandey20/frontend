Block And Inline element

Block

Element hmesa pura width occupy karte hai and new line se start hote hai

  • Always start from a new line
  • Take full width
  • next element new line me chle jate hai
  • height and width set kar sakte hai

Example

  • div
  • p
  • h1-h6
  • ul, li, ol,
  • section, header
  • footer etc.

inline element

inline element utni hi jgah lete jitni unki jrorat hoti hai

  • do not start a new line
  • take only required width
  • height and width set nahi kar sakte (mostly ingnored)
  • onlily left and right margin work karti hai
  • same line me next elelment aa jata heading

Example

  • span
  • a
  • strong, em
  • img(special case inline block)
  • label, small, etc.
